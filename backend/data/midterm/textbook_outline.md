# Financial Statements — Teaching Spine (Seed)
- **I/S Flow:** Revenue → COGS → Gross Profit → Opex → Op Inc → Interest/Taxes → Net
- **B/S Pairs:** AR↔Revenue, Inventory↔COGS, AP↔Inventory, PPE↔Depreciation
- **Cash Flow:** CFO (Net + WC + Non-cash), CFI (Capex), CFF (Debt/Equity)
- **Margins:** Gross vs Operating vs Net
- **Working Capital:** DSO, DIO, DPO; Cash Conversion Cycle

Use this spine to shape prompts and feedback for each iteration.
